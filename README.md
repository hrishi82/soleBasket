# soleBasket
 soleBasket is a e-commerce site for shoes

 ## Tech stack used
 - HTML
 - CSS

## Pages Added
- Home Page
- Product Listing Page
- Filters by
    - Price
    - Category
    - Ratings
- Cart Management
- Wishlist Management
- Authentication
    - Sign-up
    - Login
    - Logout
